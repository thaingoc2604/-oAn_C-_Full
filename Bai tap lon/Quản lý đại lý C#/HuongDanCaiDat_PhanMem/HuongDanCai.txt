Sau đó vào Start > Program > Microsoft SQL Server >Enterprise Manager
Nhìn qua trái màn hình mới hiện nhấn Double Click vào Microsoft SqL Servers > SQL Server Group > (local)(Windows NT) 
Nhấn Chuột Phải vào Thư Mục Databases>All Tasks > Attach Database
